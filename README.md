# WebGL-hosting
Pelaa pelejä selaimessa!

1) Aseta pelisi static-kansioon
2) Käynnistä sovellus ```uvicorn app:app --host 0.0.0.0 --port 5000```
3) Valitse peli url-koodilla kuten ```http://localhost:5000/01```
